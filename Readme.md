# convert pcm file to wave

```
$ ./PCM2Wave.exe

usage: ./PCM2Wave.exe in.pcm out.wav sample_rate sample_bits channels

in.pcm       : pcm file to input
out.wav      : wave file to output
sample_rate  : e.g. 8000,12000,16000,24000,32000,44100,48000...
sample_bits  : e.g. 16
channels     : e.g. 1,2...
```
